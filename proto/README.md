# ProtocolBuf Plugins
## gocosmos
```
    git clone git@github.com:regen-network/cosmos-proto.git
    cd cosmos-proto
    go install protoc-gen-gocosmos
```

## grpc-gateway
```
    git clone git@github.com:grpc-ecosystem/grpc-gateway.git
    git checkout v1.16.0
    go install ./protoc-gen-grpc-gateway/
```
